Package name:   Houdini Sanity Check Tool

Contact:        <ling.lin@intel.com>

Update:         May.24 2013


Pre-Condition:
You need have adb command avaliable in your path
Additionally, for Linux machine, you need have dos2unix avaliable in your path


Usage:
Make sure the device is connected to Host running this test
For android ICS:
./check_houdini.sh ics (Linux)
.\check_houdini.bat ics (Windows)

For android JellyBean 4.1
./check_houdini.sh jb41 (Linux)
.\check_houdini.bat jb41 (Windows)

For android JellyBean 4.2
./check_houdini.sh jb42 (Linux)
.\check_houdini.bat jb42 (Windows)
